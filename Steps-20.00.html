Creating Waterfall table 20


  • In this step we will create a waterfall table by using the sub kpi measures then switch it to a waterfall measure
  • On the visual screen in the home tab of power bi ribbon click on the enter data button then a pop up window will appear 
  • Add a ID column and add id 1,2,3,,4 for ease of creating the waterfall measure 
  • Add  a measure column where we will put our  sub kpi measure  as they appear in the measure table P_Diff%, HS YoY%, MRI YoY%, CT YoY%, HB YoY% , on the bottom name the table waterfall then save
  • Click on the water fall you have created and select new measure to create a waterfall measure and insert the DAX formula below 
Waterfall measure =
SWITCH (
    SELECTEDVALUE ('Waterfall_Hospital'[ID]),
    1,[P_Diff%],
    2, [HS YoY%]*-1,
    3, [MRI YoY%],
    4, [CT YoY%],5,[HB YoY%])
Then click the check mark
