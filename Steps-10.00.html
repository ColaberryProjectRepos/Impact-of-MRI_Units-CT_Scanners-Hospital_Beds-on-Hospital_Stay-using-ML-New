Creating Refresh Date table and My Measures table 10

  

  • On the home page under Get Data select Blank Query and Rename it to Refresh Date
  • On the Formula window enter this Formula Datetime.LocalNow()
  • A date time will appear
  • The on the top left change it to a table
  • Rename the Column to Last Refresh and Save
  • To Create My Measure Table, click on Enter Data on the Power BI Ribbon in the home page
  • On the pop up window select Edit
  • Change the name of the Table to My Measure then click close and apply
